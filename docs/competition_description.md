🎯 比赛概况

Vibe Hacks #04 是一场由 VibeFriends 主办的 24 小时极限编程（Vibe Coding）黑客松。我们诚邀开发者齐聚上海，在一天一夜的时间内，围绕“一分钟小游戏”的主题进行极限创造与开发。

🎮 核心主题：Vibe Coding for 1分钟小游戏

本届比赛的核心是创作“一分钟”小游戏。我们鼓励你对“一分钟”这个概念进行自由、多样且有趣的解读，例如但不限于：
•   计时挑战：游戏本身是倒计时一分钟的竞赛。

•   周期事件：游戏中每分钟会发生某个特定事件。

•   场景契合：专为摸鱼、等电梯、蹲坑等碎片化的一分钟等待时间设计。

•   技能掌握：玩家恰好能用一分钟学会的某种游戏技能。

我们尤其期待看到具备以下特质的有趣作品：
•   🌀 让人欲罢不能、想“再来一局”的抽象游戏。

•   🔀 由反常识设计带来全新体验的游戏机制。

•   🤖 巧妙结合AI技术的新型玩法。
