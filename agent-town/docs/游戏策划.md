# 《时空杂货镇》游戏策划案（Agent Town Project）

## 一、项目概述

### 游戏名称

《时空杂货镇》

---

## 二、游戏定位

### 游戏类型

- Agent 社交模拟
- AI 角色互动
- 开放式群像叙事
- 小镇生活模拟

### 核心卖点

一句话概括：

> “一个由文学与历史人物共同生活的、会自行运转的 AI 小镇。”

玩家不是救世主，而是世界中的普通居民。

整个小镇：

- 会记忆
- 会社交
- 会结盟
- 会吵架
- 会交易
- 会传播消息
- 会自行发展关系

玩家介入后，故事自然生长。

---

# 三、世界观设定

## 1. 世界背景

一座脱离时间长河的小镇。

不同朝代、神话、文学作品中的人物，因为“时空潮汐”被卷入此地。

这里没有原本的王朝与战争。

所有人物必须重新生活。

于是：

- 神仙开始摆摊
- 将军开始喝酒
- 才女开始教书
- 妖怪开始做生意

整个世界像一锅慢火熬煮的群像戏。

---

## 2. 世界规则

### 时间系统

- 日夜循环
- 四季变化
- 节日事件
- 天气变化

### 社会规则

角色需要：

- 吃饭
- 睡觉
- 工作
- 社交
- 获得资源

### 情绪规则

角色拥有实时情绪：

- 开心
- 焦虑
- 愤怒
- 疲惫
- 孤独
- 满足

情绪会影响：

- 对话
- 行为
- 交易
- 决策

---

# 四、核心玩法循环

## 核心循环

```text
探索小镇
   ↓
遇见角色
   ↓
交流互动
   ↓
建立关系
   ↓
参与事件 / 交易 / 冲突
   ↓
改变角色记忆与世界状态
   ↓
触发新的剧情与关系变化
```

# 七、NPC 数据结构

## 1. 基础信息

```json
{
  "name": "林黛玉",
  "source": "红楼梦",
  "age": 18,
  "occupation": "书院抄书人"
}
```

---

## 2. 性格系统

使用 Big Five + 自定义标签。

### 示例

```json
{
  "openness": 95,
  "agreeableness": 62,
  "extroversion": 20,
  "neuroticism": 88,
  "tags": [
    "敏感",
    "才情",
    "重情",
    "自尊心强"
  ]
}
```

---

## 3. 需求系统

角色持续产生需求：


| 类型   | 示例    |
| ---- | ----- |
| 生理   | 饥饿、疲劳 |
| 社交   | 想聊天   |
| 情感   | 希望被理解 |
| 资源   | 缺钱    |
| 长期目标 | 写诗集   |


需求驱动行为。

---

## 4. 行程系统

NPC 会自主安排生活：

```text
07:00 起床
08:00 去茶馆
10:00 抄书
13:00 午饭
15:00 散步
18:00 回家
21:00 睡觉
```

特殊事件会打断日程。

---

# 八、记忆系统（灵魂）

## 1. 短期记忆

保存最近事件。

例如：

- 玩家送了花
- 张飞吵架
- 今天下雨

---

## 2. 中期记忆

形成印象。

例如：

- “玩家说话很温柔”
- “张飞容易冲动”

---

## 3. 长期记忆

影响人生关系。

例如：

- “玩家曾救过我”
- “某人背叛过我”

---

## 4. 记忆结构

```json
{
  "event": "玩家送药",
  "emotion": "感动",
  "importance": 82,
  "participants": ["玩家", "林黛玉"],
  "timestamp": "Day12"
}
```

---

| 属性  | 含义   |

| 喜爱  | 是否喜欢 |
| 信任  | 是否放心 |

### 增加

### 降低

# 十、对话系统

## 1. 对话目标

不是“播放文本”。

而是：

- 交换信息
- 表达情绪
- 改变关系
- 推动事件

---

## 2. 对话结构

### 输入层

玩家自由输入或选项输入。

### 意图识别

系统识别：

- 询问
- 安慰
- 交易
- 调情
- 挑衅
- 求助

### Agent 决策

NPC 根据：

- 性格
- 当前情绪
- 记忆
- 关系
- 场景

决定回复。

### 输出生成

生成符合角色风格的语言。

---

## 3. 风格控制

### 林黛玉

- 委婉
- 敏感
- 文雅

### 孙悟空

- 快速
- 调侃
- 不耐烦

### 张飞

- 直接
- 豪爽
- 情绪化

---

# 二十、长期目标（真正厉害的部分）

最终目标不是：
“和 AI 聊天”。

而是：

> “观察一个由经典人物组成的社会，如何自行生长。”

玩家有时甚至只是坐在茶馆里。

看着：

- 林黛玉写诗
- 张飞喝酒
- 孙悟空和老板讨价还价

然后某天突然发现：

他们已经拥有了自己的生活。 🌙
